# Pan-cancer-analysis-of-tumor-microenvironment-subtypes-based-on-single-cell-RNA-sequencing
This repository contains the analysis pipeline and scripts for a pan-cancer study classifying tumor microenvironment (TME) subtypes based on single-cell RNA sequencing (scRNA-seq) data. We utilize unsupervised clustering and signature scoring to define distinct TME archetypes across multiple cancer types.
